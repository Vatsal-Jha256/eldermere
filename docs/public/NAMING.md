# Naming

The public name should feel mythic, modern, and extensible. It should not be too close to Evennia, TalesMUD, DnD, Pokemon, Arthur-specific branding, or any modern adaptation.

## Current Recommendation

**Eldermere**

Why it works:

- Feels close to the world of old legends without being locked to Arthur.
- Has the smooth fantasy-platform feel of names like Evennia without copying its sound too closely.
- Can contain many myth regions: Arthurian lakes, Greek seas, Norse fjords, South Asian rivers, Celtic otherworld crossings.
- Good repo name: `eldermere`.

## Strong Alternatives

- **Mythweald**: Strong connected-myth-world name, but a little more old-forest than browser/MUD platform.
- **Asterfell**: Cool and memorable; suggests fallen stars, prophecy, and many myth regions.
- **Oathmere**: Excellent for Arthurian-first, still broad enough for other legends.
- **Lorewick**: Compact, mod-friendly, and slightly strange; good if the tone leans playful/street-myth.
- **Veyr**: Short and distinctive, but more abstract and may need more branding work.
- **Morrowmere**: Strong mythic feel, but longer.
- **Runehaven**: Clear fantasy meaning, but less distinctive.
- **Mythren**: Nice sound, close to "myth" and "children/kin", but a little softer.

## Names To Avoid

- Names containing `DnD`, `D&D`, `Pokemon`, or direct references to protected brands.
- Names that imply affiliation with a film, show, game, or modern adaptation.
- Names that lock the project only to Arthur if the long-term goal is a connected legend universe.

## Repo Recommendation

Create the GitHub repository as:

```txt
eldermere
```

Use the display title:

```txt
Eldermere
```

The working subtitle:

```txt
A browser MUD for connected legends.
```
