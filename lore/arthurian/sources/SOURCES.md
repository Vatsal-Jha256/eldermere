## Use Policy

- Use these texts for lore grounding and source comparison.
- Do not copy long source passages into game content.
- Write original game prose, dialogue, scenes, mechanics, factions, and quest outcomes.
- Record source disagreements as useful variation points for probabilistic quests.
- Do not use modern adaptations as source material unless their license explicitly allows it and the project intentionally adopts that license.
